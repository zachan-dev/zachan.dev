ALM Open Test Architecture API Type Library Version 12.5x
IComponent5 Interface
Members 
Description
Represents a single Business Process Component.
Public Methods
 BusinessProcessesThe list of associated BusinessProcess objects.
 DeletePictureDeletes the snapshot.
 LockObjectLocks the object. Returns True if the object has been changed on the server.
 PostPosts all changed values into database.
 RefreshReads saved values, overwriting values in memory.
 SetStepsDataFor backward compatibility with versions prior to 9.0. Wrap component's step in HTML format.
 UndoUndoes changes to field values that have not been posted.
 UnLockObjectUnlocks the object.
 UploadSnapshotSaves the component snapshot. The snapshot file name must be 'Picture.png'.
Public Properties
 ApplicationAreaIDThe Component's linked application area ID.
 AttachmentsThe Attachments property is not implemented for the Component object or for any IComponent* interface.
 AutoPostIf true, the database is updated immediately when the field value changes.
 ComponentParamFactoryThe component parameter factory.
 ComponentStepFactoryThe ComponentStepFactory for this component.
 ExtendedStorageThe ExtendedStorage object for this component.
 FacetFactoryThe FacetFactory for this component.
 FieldThe value of the specified field.
 FolderThe component's folder.
 HasAttachmentChecks if the object has one or more attachments.
 HasPictureIndicates if the component has a snapshot.
 HasScriptChecks if the component is associated with a test script.
 IDThe item ID.
 IsIteratableFor backward compatibility with versions prior to 9.0.
 IsLockedChecks if object is locked for editing.
 IsObsoleteChecks if the component has been marked deleted by a user but is still in use in a test.
 ModifiedChecks if the item has been modified since last refresh or post operation. If true, the field properties on the server side are not up to date.
 NameThe Component's name.
 ScriptTypeThe test script type.
 VirtualChecks if this is a virtual item, that is, an item that does not have a corresponding database record.
