ALM Open Test Architecture API Type Library Version 12.5x
IRun Interface
Members 
Description
Represents a test run.
Public Methods
 CancelRunCancels the run.
 CopyDesignStepsCopies design steps into the test run of an executed test.
 CopyStepsToTestCopies all run execution steps, including new added steps, into the design steps of the corresponding planning test.
 LockObjectLocks the object. Returns True if the object has been changed on the server.
 PostPosts all changed values into database.
 RefreshReads saved values, overwriting values in memory.
 ResolveStepsParametersUpdates the texts of the run's steps by resolving the parameter values at run time.
 UndoUndoes changes to field values that have not been posted.
 UnLockObjectUnlocks the object.
Public Properties
 AttachmentsThe Attachment factory for the object.
 AutoPostIf true, the database is updated immediately when the field value changes.
 ExtendedStorageThe ExtendedStorage object for the current run.
 FieldThe value of the specified field.
 HasAttachmentChecks if the object has one or more attachments.
 IDThe item ID.
 IsLockedChecks if object is locked for editing.
 ModifiedChecks if the item has been modified since last refresh or post operation. If true, the field properties on the server side are not up to date.
 NameThe run name.
 ParamsDeprecated. Use ParameterValueFactory of ISupportParameterValues. The step parameters of this run.
 ResultLocationThe run result location, a UNC path.
 StatusThe run status.
 StepFactoryThe Step Factory for the current run.
 TestIdThe test ID of the test that has been run.
 TestInstanceThe number of this instance of the design test in the test set.
 TestSetIDThe ID of the test set to which the run belongs.
 VirtualChecks if this is a virtual item, that is, an item that does not have a corresponding database record.
