ALM Open Test Architecture API Type Library Version 12.5x
IPolicyStatus3 Interface
Members 
Description
The Policy Enforcement Status information of the connected project.
Public Methods
 RefreshRefreshes policy status data.
Public Properties
 AdministratorsThe list of project administrators.
 BlockPolicyUpdateIndicates whether there is a request to block propagation to the project.
 DomainNameThe project's domain name.
 IsActiveIndicates whether the project represented by this policy status is active or not.
 IsTemplateChecks whether the connected project is a template.
 IsUpToDateChecks whether the connected project is up-to-date with its linked template.
 LastPropagationReportThe XML document containing the data of the last propagation performed on the project.
 LastPropagationTimeThe date-time of the last template propagation performed on the project.
 LastVerificationReportThe XML document containing the data of the last verification performed on the project.
 LastVerificationTimeThe date-time of the last template verification performed on the project.
 LinkedTemplateThe Policy Enforcement Status object containing details of the project's linked template.
 NameThe project's name.
 PolicyUpdateBlockCommentThe policy enforcement block comment.
 VerificationStatusChecks whether the project is verified against its linked template.
