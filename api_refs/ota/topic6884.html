ISupportRBTAssessment2 Interface
Members 
Description
Represents Risk-based Quality Management assessment capabilities for the Req coclass.
Public Methods
 AssessBILevelAssess the Business Criticality risk level for this requirement, according to the values provided.
 AssessFCLevelAssess the Functional Complexity risk level for this requirement, according to the values provided.
 AssessFPLevelAssess the Failure Probability risk level for this requirement, according to the values provided.

 AssessedFCLevelFunctional Complexity risk level for this requirement, from the last time we called AssessBILevel.
 AssessedFPLevelFailure Probability risk level for this requirement, from the last time we called AssessFPLevel.

 CalculatedRiskLevelCalculated Risk level for this requirement.

 CustomFPLevelCustom Failure Probability level, overrides the assessed level.
 CustomRiskLevelCustom Risk level, overrides the assessed level.
 EffectiveBILevelEffective Business Criticality level.
 EffectiveFCLevelEffective Functional Complexitylevel.
 EffectiveFPLevelEffective Failure Probability level.
 EffectiveRiskLevelEffective Risk level.
 FCChosenAnswerForQuestionThe chosen value to this Functional Complexity criterion for this requirement.
 FPChosenAnswerForQuestionThe chosen value to this Failure Probability criterion for this requirement.

 UseCustomFCLevelDoes this requirement have a custom Functional Complexity level?
