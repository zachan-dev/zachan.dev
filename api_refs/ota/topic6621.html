ALM Open Test Architecture API Type Library Version 12.5x
ISettings2 Interface
Members 
Description
Represents users' settings in various user-defined categories.
Public Methods
 CloseCloses and updates the category.
 DeleteCategoryDeletes the current settings folder.
 DeleteValueDeletes the specified item.
 OpenSets the category to be used in subsequent actions by the client.
 PostPosts the category to the server.
 RefreshReads saved values, overwriting values in memory.
Public Properties
 EnumItemsReturns the list of names of the setting's items.
 IsSystemChecks if the specified setting is built-in and read-only.
 ValueThe value of the specified item in the active category.
