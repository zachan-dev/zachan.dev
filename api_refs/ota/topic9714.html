UserAsset Object
Members 
Description
Represents any entity known to the testing tool that is not otherwise represented in Quality Center.
Remarks

A user asset is an entity used to represent assets of a testing tool. It can represent an abstraction of the testing tool or a tangible asset, such as a file.

A user asset is owned by another ALM entity of a type that implements ISupportAssets, for example, Component, Test, and QCResource. 

A user can add, delete, or modify a user asset if the user has Modify permissions on the entity that owns the user asset.

Get a reference from UserAssetFactory.
