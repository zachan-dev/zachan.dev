Public Methods
 RunStarts execution the test set or of the specified tests.

Public Properties
 ChangeDetectEnabledIndicates whether change detection mode is enabled.
 ExecutionLogThe execution error or log message sent by the remote agent.
 ExecutionStatusThe execution status object for the scheduler execution.
 HostTimeOutThe time to wait for response before either sending test to another host in group or failing.
 LogEnabledIndicates if logging is enabled.
 RunAllLocallyIndicates if all tests are to be run on the local host.
 RunOnHostThe host on which to execute the test instance.
 SkipNonChangeDetectableComponentsEnabledIndicates whether to skip components that are not change detectable.
 TdHostNameThe host on which to run all tests in the current scheduler execution.
 VM_ConfigObsolete. Do not invoke.
