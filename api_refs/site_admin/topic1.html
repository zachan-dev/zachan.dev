ALM Site Administration API Reference (12.50 and later)
What's new in this version

In version ALM 11 Service Pack 3 and later versions, the SAClient Type Library can no longer be registered on your workstation by running the Site Administration Client. You must run the Client Registration add-in. The add-in is on the ALM Tools page, accessed from the help drop-down menu in ALM.  For more details, see the information about registering on a workstation in the Application Lifecycle Management Installation Guide.
