td.COMPONENT_STEP Table
Project Data Tables Database > td Schema : td.COMPONENT_STEP Table
Description
Steps of Business Process Components.
Remarks

In the description and expected result strings, parameters are enclosed in triple angle brackets. The HTML representation is &lt;&lt;&lt;PARAMETER NAME&gt;&gt;&gt;

Parameters are linked to steps in the COMPONENT_STEP_PARAMS Table.
