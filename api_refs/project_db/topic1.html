ALM Project Database Reference (12.50 and later)
Introduction

This is a reference for Application Lifecycle Management (ALM) project databases. The reference is based on a project hosted by a SQL Server.

The usage of the project data base may change from version to version in response to changes in ALM functionality and implementation. Tables and fields may be added or dropped, and the use made of them may change. Micro Focus does not commit to maintaining compatibility at the data level between versions of ALM.

This reference covers only permanent data tables present after any installation.  See Undocumented Tables.

The tables and fields are identical for SQL Server and Oracle databases. However, there may be differences in other items, such as triggers and views. If you have questions specific to projects hosted on an Oracle database, visit our support site.

This reference includes diagrams that reflect focused areas of interest. See Functional Groups Diagrams.
