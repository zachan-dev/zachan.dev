ALM 12.5x REST API Reference
Code Samples

These examples of JavaScript code show how to perform basic functions of the REST API. You can study the logic, then implement in the programming language of your choice.

The examples can be run if you want. To run them:

  • Make a folder under qcbin on the server.
  • Use the Copy Code button to save the text of the "HTML File to run examples" to the clipboard. Paste the text into a text editor and save as HTML to your folder under qcbin.
  • Do the same for each of the JavaScript examples. Name the files according to the name in the comment at the top of each file.
  • Run the HTML file in a browser.
