Instances
Working with the API > Instances

An instance is a resource that represents a single member of a collection.  For example, one defect, one lock, and so on.

The URI of an instance is the URI of the containing collection with the ID of the instance appended.  The format is: https://{server}[:port]/qcbin/api/domains/{domain name}/projects/{project name}/{collection name}/{ID}.

For example: https://myserver:8080/qcbin/api/domains/DEFAULT/projects/PEMBERLY/defects/100

Instances support HTML methods:

  • GET
  • DELETE
  • POST
