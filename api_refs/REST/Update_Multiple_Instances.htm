Update Multiple Instances

To update more than one instance, PUT data for more than one instance.

Put defects
Copy Code 
*** Request ***
PUT /qcbin/api/domains/{domain}/projects/{project}/defects
Accept:             application/json
Host:               api.alm.com:8080
