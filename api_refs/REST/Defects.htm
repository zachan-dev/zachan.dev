ALM 12.5x REST API Reference
Defects
Reference > REST Resources in this API > Defects

The defect collection contains defect instances. Each instance represents an ALM Defect.

Remarks

None

URIs

/qcbin/api/domains/{domain}/projects/{project}/defects

/qcbin/api/domains/{domain}/projects/{project}/defects/{ID}

Limitations

  • See General Notes and Limitations

  • Some defect fields cannot be used for ordering or filtering. They are:

    dev-comments, cycle-id, description, has-linkage, has-others-linkage, request-note, request-type, request-server, run-reference, step-reference, test-reference, cycle-reference, ver-stamp
