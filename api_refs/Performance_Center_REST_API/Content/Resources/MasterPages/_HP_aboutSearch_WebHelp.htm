Search Tips

Search the online help using the Search pane in the left sidebar.

  • To search for an exact phrase, surround the words with quotes. Example: “release backlog”

  • If you enter a string of words without quotes, the results return topics that include any of the searched words.

  • To limit your search to a specific area, select a category in the Search filters box.

  • To save a search query for future reference, click in the Search pane.
    To access your saved queries, open the Favorites pane in the left sidebar.

Use the following expressions (or combinations of them) to refine your search results:

Search Expression Results

release and management

release+management

Returns topics that include both the words “release” and “management”, but not necessarily as a phrase.

not “release management”

! ”release management”

Returns topics that do not include the phrase “release management”.
release ^ management Returns topics that include the word “release” but not the word “management”
