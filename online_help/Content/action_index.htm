Dashboard > Analysis View actions

Action Description
Generate

Generates the selected analysis item.

Graphs: The results are displayed in the View tab. If the View tab is already selected, the analysis item is refreshed with the most up-to-date data.

Project Reports and Excel Reports: The results are displayed in an external application.
Graph Wizard Opens the graph wizard, enabling you to create and configure a graph.
New Business View Graph

Opens the New Business View Excel Report dialog box, enabling you to create a new business view Excel report in the analysis tree.
New Graph Opens the New Graph dialog box, enabling you to create a new graph in the analysis tree.
New Business View Excel Report Opens the New Business View Excel Report dialog box, enabling you to create a new business view Excel report in the analysis tree.
New Excel Report Opens the New Excel Report dialog box, enabling you to create a new Excel report in the analysis tree.
New Project Report Opens the New Project Report dialog box, enabling you to create a new project report in the analysis tree.
New Release Analysis

Creates a new graph in the analysis tree. The following options are available:

  • Release Scorecard. Enables you to Opens the New Graph dialog box, enabling you to create a new Release Scorecard graph in the analysis tree.
  • KPI Over Time Graph. Opens the New Graph dialog box, enabling you to create a new KPI Over Time graph in the analysis tree.
  • KPI Breakdown Graph. Opens the New Graph dialog box, enabling you to create a new KPI Breakdown graph in the analysis tree.
  • KPI Breakdown Over Time Graph. Opens the New Graph dialog box, enabling you to create a new KPI Breakdown Over Time graph in the analysis tree.

Dashboard > Dashboard View actions

Action Description
Generate

Generates the selected dashboard page, and displays it in the View tab.

If the View tab is already selected, the dashboard page is refreshed with the most up-to-date data.
Export

Saves the selected dashboard page as a PDF file, enabling you to print the dashboard page. The following options are available:

  • Dashboard Page Layout. The dashboard page is saved in its original format.

  • One Item Per Page Layout. Each graph on the dashboard page is saved on a separate page.

    Available from: View tab

Management > Releases actions

Action Description
Details Displays details of the currently selected folder, release, or cycle. The Statistics area displays the requirements and test set folders associated with the release or cycle.
New Release Opens the New Release dialog box, enabling you to add a release to a selected release folder.
New Milestone Opens the New Milestone dialog box, enabling you to add a milestone to a selected release.
New Cycle Opens the New Cycle dialog box, enabling you to add a cycle to a selected release.
Reschedule Opens the Reschedule dialog box, enabling you to reschedule the start and end date of a release, cycle, or milestone.

Management > Libraries actions

Action Description
Create Library Opens the Create Library dialog box, enabling you to add a library to a selected library folder.
Import Library Opens the Import Library Wizard, enabling you to import a library under a selected library folder.
Create Baseline Enables you to create a baseline for a selected library.
Compare To

Compares the currently selected library or baseline with other libraries or baselines. Includes the following options:

For a selected library:

  • Latest Baseline Created. Compares the selected library with the most recent baseline created in the other library.
  • Select Baseline. Opens the Select Baseline dialog box, enabling you to select a baseline in another library with which to compare.

For a selected baseline:

  • Current Entities. Compares the selected baseline with the current entities in the library.
  • Select Baseline. Opens the Select Baseline dialog box, enabling you to select a baseline in the library with which to compare.
Library Details Opens the Library Details dialog box, enabling you to view and edit the details of the selected library.

Mark as Template Test

For manual tests. Marks a test as a template test that you can then call from another test. After you mark a template test, the manual test icon changes from gray to white .

For GUI tests. Marks a test as a template test that you will then be able to use to create other UFT GUI tests.

Testing > Test Lab actions

Action Description
New Test Set Opens the New Test Set dialog box, enabling you to add a test set to a selected folder.
Go to Test Set Opens the Go to Test Set dialog box, enabling you to type a Test Set ID to find a specific test set in the test sets tree. To locate the Test Set ID for a test set, select the test set in the test sets tree and click the Details tab.
Pin to Baseline Opens the Select Baseline dialog box, enabling you to pin the test set to a baseline.
Import External Test Results Opens the Import External Test Results dialog box, enabling you to import a results file generate from an external tool to ALM.
Assign to Cycle Enables you to assign a test set folder to a cycle.
Select Tests

Enables you to select tests to add to the currently selected test set. Displays the following:

  • Test Plan Tree tab. Add tests from the test plan tree to a test set.
  • Requirements Tree tab. Add tests covering requirements from the requirements tree to a test set.
Run Test Set

For Default test sets:

  • If the test set includes one or more automated tests, opens the Automatic Runner dialog box to run the currently selected test set.
  • If the test set includes manual tests only, opens the Manual Test Run dialog box.

For Functional test sets: Opens the Run <Entity> Dialog box, enabling you to run the selected functional test set.

Reset Test Set

Opens the Confirm Reset Test Set dialog box, enabling you to change the status of all the tests in the test set to No Run.

You can also instruct ALM to delete all the test run results in the test set.

Purge Runs Opens the Purge Runs wizard, enabling you to delete old test run results in your project.
Test Set Details Opens the Test Set Details dialog box, enabling you to view and update details of the selected test set.
Host Manager

For Default test sets: Opens the Host Manager dialog box, enabling you to define hosts for remote test execution within a Default test set.

For Functional test sets: Opens the Testing Hosts module, enabling you to manage and define hosts for server-side Functional test set execution.
Export Exports the test set data as a text file, Microsoft Excel worksheet, Microsoft Word document, HTML document, or text document.

Testing > Test Runs actions

Action Description
Send by E-mail Opens the Send E-mail dialog box, enabling you to send test runs by email to recipients selected from a list or to the tester.
Details Opens the Run Details dialog box, enabling you to view and edit details of the selected run.
Continue Manual Run Continues running the selected tests with the same runner that was used the first time the test was run.
Purge Runs Opens the Purge Runs wizard, enabling you to delete old test run results in your project.

Defects actions

Action Description
New Defect Opens the Create New Defect dialog box, enabling you to add a defect, or continue editing a defect that you started to add.
Go to Defect Opens the Go To Defect dialog box, enabling you to find a specific defect by Defect ID.
Send by E-mail Opens the Send E-mail dialog box, enabling you to send a defect by email to recipients selected from a list.
Defect Details Opens the Defect Details dialog box, displaying the details of the selected defect.
Export

Opens the Export Grid Data dialog box, enabling you to export the defects in a grid as a text file, Microsoft Excel worksheet, Microsoft Word document, or HTML document.

Select one of the following options:

  • All. Exports all defects in the grid.
  • Selected. Exports selected defects in the grid.
